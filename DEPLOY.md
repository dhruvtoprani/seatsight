# Deploy SeatSight to Vercel

The app is a static frontend and is ready for Vercel.

## Fastest path

1. Go to Vercel.
2. Import `dhruvtoprani/seatsight`.
3. Use the default static deployment settings.
4. Deploy.

No build command is required. The output directory can be left as the repository root.

## Local test

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.
